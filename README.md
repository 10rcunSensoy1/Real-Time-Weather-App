# Real-Time-Weather-App
This is a Kotlin-based weather app that we're developing as a team. It builds upon an existing open-source project with improvements and new features.
🚀 Features:

Real-time weather updates

City-based forecasts

Modern UI with Jetpack Compose

API integration for accurate data

🎯 Goals:

Enhance the app with new functionalities

Improve our Kotlin & Android development skills

Contribute to the open-source community

📢 Contributions:
Feel free to submit issues and pull requests!

🔗 Tech Stack: Kotlin, Retrofit, MVVM, Jetpack Compose
